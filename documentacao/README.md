# 📚 Documentação do SIGEDAM

Este diretório contém toda a documentação do projeto **SIGEDAM**.

## Objetivo
A documentação tem como propósito centralizar informações técnicas e funcionais do sistema, servindo como guia para desenvolvedores, analistas, professores e demais colaboradores.  
Aqui devem estar descritos processos, arquiteturas, decisões de projeto, instruções de instalação, uso e manutenção.

## Estrutura
A organização da pasta deve seguir a divisão por temas ou responsabilidades, por exemplo:

- `exemploDePasta/` — Descreve que tipo de documentação esse diretório irá conter  
- `OutroExemplo/` — Deve ser feito para cada diretório 
- `OutroExemplo/usuario` — Também deve ser feito para cada subpasta


## ⚠️ Aviso Importante
Sempre que **novas subpastas ou documentos forem criados**, o **README.md deve ser atualizado** para refletir a estrutura atual e facilitar a navegação.
